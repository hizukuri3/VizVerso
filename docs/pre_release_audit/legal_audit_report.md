## 2. 知的財産 (IP) 保護とデータ秘匿性

計算式（知的財産）および解析データの外部流出がないことを技術的に検証しました。

- **テレメトリ排除**: `Sentry`、`Google Analytics`、`LogRocket` 等の追跡・監視ライブラリは一切導入されていません。
- **通信スキャン結果**: `src` ディレクトリ内を網羅的にスキャンした結果、`fetch` や `XMLHttpRequest` による外部 API へのデータ送信コードは存在しません。
- **外部スクリプトの制限**:
  - `index.html` にて唯一読み込まれている外部スクリプトは `Buy Me a Coffee` ウィジェットのみです。
  - このスクリプトはウィジェット描画のみを行い、ワークブックの内容（計算式等）にアクセスするコードは含まれていません。

## 3. クライアントサイド完結の保証

本アプリケーションは、**100% ブラウザ内（サンドボックス）**で動作することを保証します。

- **パース処理**: Web Worker（`tableauParser.worker.ts`）を利用し、ブラウザのバックグラウンドスレッドでローカルファイルを解析しています。
- **サーバーレス**: サーバーサイドの解析処理（Node.js, Python等）を必要とせず、静的な Web サイトとしてホスティング可能です。
- **データ保護**: ユーザーがアップロードしたファイルはメモリ内でのみ保持され、ブラウザを閉じれば完全に消去されます。

## 4. 依存関係の最小化

リーガルリスク低減のため、以下の取り組みを実施しています。

- **標準APIの優先**: ハイライト処理や文字列操作において、サードパーティライブラリを避け JavaScript 標準の `RegExp` および `String` API を優先して使用。
- **軽量化**: プロジェクトの目的（Tableau解析）に直結しない「便利ライブラリ（lodash, moment等）」を排除し、フットプリントを最小限に抑えています。

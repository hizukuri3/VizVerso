# 高度な検索機能の実装完了報告

計算フィールドの依存関係まで考慮した、強力な検索ナビゲーション機能を実装しました。

## 実施内容

### 1. 依存関係を考慮した検索エンジン
`useSearch` フックを新規実装し、以下のロジックを実現しました：
- **名称一致**: シート、データソース、フィールド名の部分一致。
- **計算式一致**: 計算フィールドの式内に検索語が含まれる場合にヒット。
- **連鎖ヒット (Dependency Match)**: ヒットしたフィールドを計算式内で参照している「親」フィールドや、そのフィールドを使用している「ワークシート」を自動的に抽出。これにより、「この項目を変更するとどこに影響が出るか」を検索から即座に把握できます。

### 2. インテリジェントな UI
ヘッダーを刷新し、以下の機能を備えた検索バーを導入しました：
- **ヒット理由の可視化**: 「直接一致」「計算式」「依存関係」など、なぜそのアイテムがヒットしたかをバッジで表示。
- **スマートナビゲーション**: 
  - フィールドがヒットした場合、その「定義元」であるデータソース画面へ遷移。
  - シートがヒットした場合、その「詳細ビュー」へ遷移。
- **高速なフィードバック**: メモ化されたインデックスにより、大規模なワークブックでも入力と同時に結果が表示されます。
